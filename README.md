# Учебный Python проект

## Описание:

Проект для изучения языка Python.

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/DenisBoiarinoff/study_bank_operations_widget.git
```

2. Установите зависимости:
```
poetry install
```

## Использование:

Склонипркйте проект, установите зависимости и эксперементируйте в `main` модуле


## Документация:

Пакет src содержит модули 
 - `masks` - функции маскирования номеров считов и карт
 - `widget` - функция преодразования даты из iso 8601 в '%d.%m.%Y'
 - `processing`  - функции соритровки и фильтрации списка словарей.
 - `generators` - функции создающин генераотры - генератор номеров карт и генеоаторы по списку транзакций.
 - `decorators` - декоратор для логирования работы функций
 - `utils` - чтение информации о транзакцияз из файла и перевод валюты транзакции в рубли
 - `external_api` - получение текущего обменного курса
